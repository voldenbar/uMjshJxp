# 项目简介

## 核心功能
- 用户管理：注册、登录、信息更新、密码管理、用户列表及批量操作
- 服务人员管理：创建、更新、详情查看、列表展示、删除与项目分配
- 服务分类管理：分类创建、更新、详情、树形结构展示、删除及批量操作
- 服务项目管理：服务创建、更新、详情、列表展示、状态管理及删除
- 订单管理：创建、状态更新、详情查看、列表展示、取消、删除及批量操作
- 评价系统：创建、删除、详情查看、列表展示
- 权限管理：角色创建、更新、删除、列表展示、菜单分配
- 菜单管理：创建、更新、删除、树形结构展示、权限分配
- 轮播图管理：创建、更新、删除、列表展示
- 收藏功能：添加、取消、收藏列表
- 浏览历史：记录、清除、历史列表
- 数据统计：系统概览、订单趋势分析
- 推荐系统：用户推荐、服务推荐、内容推荐
- 搜索功能：服务搜索、人员搜索、全局搜索
- 支付系统：模拟支付、取消支付、状态管理
- 退款管理：申请、审核、详情查看、列表展示
- 文件管理：上传功能

## 适用场景
家政服务平台的日常运营管理，包括用户与服务人员管理、服务分类与项目维护、订单处理与评价系统、权限控制与内容管理，以及数据分析与推荐功能，适用于各类家政服务企业进行数字化管理。

## 技术特点
基于SpringBoot3+Vue3全栈架构，后端采用SpringBoot、MyBatisPlus、JWT、SpringSecurity技术栈，前端使用Vue3、Element Plus、Node.js、Axios、Pinia，数据库为MySQL8，实现高性能、安全可靠的家政服务管理系统。
